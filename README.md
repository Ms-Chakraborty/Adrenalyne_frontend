# 🎫 Adreanalyne Frontend

Adreanalyne Frontend is a **React + TypeScript web application** that serves as the user interface for the Adreanalyne event ticketing platform.  
It allows users to browse events, view event details, and interact with the backend APIs for event and ticket management.

---

## 🚀 Features

- Browse published events
- View event cards and event details
- Responsive and modern UI
- Pagination for event listings
- Protected routes for authenticated pages
- Reusable UI component system

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS / UI Components
- ESLint & Prettier
